# Installation

1. Get a free API Key at [Bot Father](https://t.me/BotFather)
2. Clone the repo
   ```sh
   git clone https://github.com/ta-verma/Quotes-Telegram-Bot.git
   ```
3. Install packages
4. Enter your API in `server.py` and `quotes.py`
